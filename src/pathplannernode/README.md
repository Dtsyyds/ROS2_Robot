# pathplannernode

## 🗺️ 功能描述
`pathplannernode` 是系统的视觉路径规划模块。它负责处理 3D 点云数据，生成避障路径，并针对特定任务（如壁面作业）生成注意力路径。

## 🛠️ 核心功能
- **点云处理**: 基于 Open3D 的滤波、分割和特征提取。
- **路径生成**: 注意力路径算法，优化机器人作业轨迹。
- **坐标变换**: 实现局部坐标系与全局坐标系的实时转换。

## 📂 项目结构
- `src/pathplanner/`: 核心算法实现（Python）。
- `src/pathplanner_ros/`: ROS 2 节点接口。
